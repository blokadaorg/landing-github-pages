Blokada Slim v4.6

നിങ്ങളുടെ ബ്ലോക്കഡ അപ്‌ഡേറ്റുചെയ്‌തു!

ബ്ലോക്കഡയുടെ രണ്ട് പതിപ്പുകൾ ഉണ്ട്. നിങ്ങൾ ഉപയോഗിക്കുന്നതിനെ Google Play- യിൽ ലഭ്യമാണ്, ഇതിനെ ബ്ളോകഡ സ്ലിം എന്ന് വിളിക്കുന്നു (പഴയ പേര്: ബ്ലോക്കഡ DNS ചേഞ്ചർ). ഇതൊരു അഡ്‌ബ്ലോക്കറല്ല. നിങ്ങളുടെ ഡി‌എൻ‌എസ് സെർ‌വറുകൾ‌ മാറ്റുന്നതിനും ഞങ്ങളുടെ പുതിയ വി‌പി‌എൻ‌ സേവനം ഉപയോഗിക്കുന്നതിനും ഇത് അവസരമൊരുക്കുന്നു. ബ്ലോക്കഡയുടെ ഓപ്പൺ സോഴ്‌സ് പതിപ്പ് ഞങ്ങളുടെ വെബ്‌സൈറ്റിൽ ലഭ്യമാണ്.

The 4.6 release is a maintenance update with improvements for stability, and user experience. മാറ്റങ്ങളെക്കുറിച്ചുള്ള കൂടുതൽ വിവരങ്ങൾക്ക് ഞങ്ങളുടെ ചേഞ്ചലോഗ് പരിശോധിക്കുക.
